# IBM HR Analytics Dashboard

## Project Overview

This project is an interactive HR Analytics Dashboard developed using Tableau.

The dashboard helps analyze employee attrition, workforce distribution, and HR trends.

## Dashboard Features

- Employee Count KPI
- Attrition Count KPI
- Attrition Rate KPI
- Employees by Department
- Attrition by Department
- Attrition by Gender
- Employees by Age Group
- Employees by Education Field
- Interactive Filters

## Tools Used

- Tableau
- IBM HR Employee Attrition Dataset
